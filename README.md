# application-NetworkManager
Slate NIC, Console, Route Manager, and File Transfer

Until we build an installer, initial Win11 comp setup will require a few dependencies.

***INSTALL DEPENDENCIES***
Run these in command prompt or powershell as admin
```
pip install pyinstaller pyserial paramiko
```


***RUNNING THE APPLICATION***
1. Put main.py in a folder on your computer
2. Run command prompt or powershell as admin
3. Navigate to the main.py
4. run the app via: ``` python main.py ```
